Reccueil de Patchs Pure Date et de code GLSL 

Ce dépot de fichiers a pour objet de constituer un reccueil de shaders à utiliser dans Pure Data Gem
Les fichiers sont issus :

Atelier_Pure-Data_Gem_GLSL/
- d'un atelier réalisé avec Jack de Rybn à Labomedia début 2012 centré sur la création de shaders

Exemples_GLSL/
- d'échanges sur le forum codelab.fr, du portage de shaders provenant de sources diverses (voir http://codelab.fr/2897)

Customs/
- de créations artisanales
